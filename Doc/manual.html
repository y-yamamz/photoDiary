PhotoDiary 使い方ガイド

ログイン

ユーザー名とパスワードを入力してログイン。

アルバム画面

ログイン後のメイン画面です。

  • 左サイドバー（スマホはカレンダーアイコン）：年・月・日をクリックして絞り込み
  • ヘッダー：グループ絞り込み・各機能へのボタン
  • 写真をクリック：拡大表示。左右矢印で前後移動。詳細の編集・削除も可能

写真をアップロードする

ヘッダーの アップロード ボタン →

  1. 写真をドラッグ＆ドロップ（またはクリックして選択）※複数可
  2. グループ・撮影日・場所・説明を入力（任意）
  3. アップロード ボタンをクリック

複数まとめて操作する

ヘッダーの 選択 ボタン → 写真をチェック → 画面下のバーから操作

操作 内容
一括削除 選択した写真をまとめて削除
一括編集 場所・説明・グループをまとめて変更
ダウンロード ZIP形式でまとめてダウンロード

グループを管理する

ヘッダーの グループ ボタン →

  • 追加：グループ名を入力して「追加する」
  • 編集・削除：各グループ行の鉛筆 / ゴミ箱アイコン

掲示板

ヘッダーの 掲示板 ボタン（未読があるとバッジ表示）

タブ 内容
お知らせ 管理者からのお知らせ（未読は青くハイライト）
管理者へ連絡 件名・内容を入力して送信。返信や対応状況も確認できる

ログアウト

ヘッダーの ログアウト ボタン

注意：削除操作は元に戻せません。
